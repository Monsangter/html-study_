Chap image

Image can improve the design and the appearance 
Syntax = <img src="" alt="">

Used to embed an image in a webpage
Not technically inserted into a web page; linked to web pages.

Creates a holding space for the referenced image
Is empty
Has two required attributes:
Src - specifies the path
Alt - when in circumstances ;slow connection, error in src or user uses a screen reader.
Screen reader provide listen to the content.
Style

<img src="img_girl.jpg" alt="girl in a jacket" style="width:500px; height:600px;">
<img src="img_girl.jpg" alt="Girl in a jacket" width="500" height="600">
일케해더 픽셀단위로 인식

Make sure that the image actually stays in the same spot in relation to the web page.
Otherwise your visitors will get a broken link icon.

Width height style are all valid in html.

But using style prevents styles sheets from changing the size of images

Ex)
<!DOCTYPE html>
<html>
<head>
<style>
img { 
  width: 100%; 
}
</style>
</head>
<body>

/* The first image uses the width attribute (set to 128 pixels), but the style in the head section overrides it, and sets the width to 100%. */
<img src="html5.gif" alt="HTML5 Icon" width="128" height="128">
/* The second image uses the style attribute to set the width to 128 pixels, this will not be overridden by the style in the head section: */
<img src="html5.gif" alt="HTML5 Icon" style="width:128px;height:128px;">

</body>
</html>
따로 스타일을 태그에서 지정해주지 않으면. 스타일에 정의된대로 오버라이드!

Image floating
Let the image float to the right or to the left of a text.

<p><img src="smiley.gif" alt="Smiley face" style="float:right;width:42px;height:42px;">
The image will float to the right of the text.</p>

most common image file types, which are supported in all browsers (Chrome, Edge, Firefox, Safari, Opera):

APNG	Animated Portable Network Graphics	.apng
GIF	Graphics Interchange Format	.gif
ICO	Microsoft Icon	.ico, .cur
JPEG	Joint Photographic Expert Group image	.jpg, .jpeg, .jfif, .pjpeg, .pjp
PNG	Portable Network Graphics	.png
SVG	Scalable Vector Graphics	.svg

Use the HTML <img> element to define an image
Use the HTML src attribute to define the URL of the image
Use the HTML alt attribute to define an alternate text for an image, if it cannot be displayed
Use the HTML width and height attributes or the CSS width and height properties to define the size of the image
Use the CSS float property to let the image float to the left or to the right

Chap Image maps

Is an image with clickable areas. Defined with one more <area> tags.

To create an image. You need an image and some html code that describes the clickable areas.

1.
Usemap attribute.
<img src="workplace.jpg" alt="Workplace" usemap="#workmap">

Usemap value starts with a hash tag # flowed by the name of the image map.
Used to create a relationship between the image and the image map.

2.
<map> element is used to create an image map. And is linked to the image by using the required name attribute:
<map name="workmap">
Must have the same value as the <img>'s use map attribute.


3. Add the clickable areas.
Shape.
You can choose one of these values:
Rect - defines a rectangular region
Circle - defines a circular region
Poly - defines a polygonal region
Default - defines the entire region

Coordinate.
Co "with together" + ordination "arrangement"
v. 조직화 하다. 조정하다. 
n. 좌표. 함꼐 잘 어울리는 옷들.


<area shape="rect">
Come in pairs. From left margin, and from the top. (a, b)
From left margin and from the top (c, d)

<img src="workplace.jpg" alt="workplace" usemap="#workmap" width="400" height="379">
<map name="workmap">
<area shape="rect" coords="34,44,270,350" alt="computer" href="computer.htm">
</map>


<area shape="circle">
First locate the coordinates of the center of the circle.
Specify the radius of the circle

<image src="workplace.jpg" alt="workplace" usemap="#workmap" width="400" height="400">
<map name="workmap">
	<area shape="circle" coords="337,300,44" alt="Cup of coffee" href="coffee.htm">
</map>

Chap background image on a html element

Use html style attribute and the css background-image property

<p style="background-image: url('img_girl.jpg');">

<style>
P {
	background-image: url('img_girl.jpg');
}
</style>

Only available for paragraph

<style>
Body {
	background-image: url('img_girl.jpg');
}
</style>

Available for entire body

!back ground repeat

If bg image is smaller than the element. Image will repeat itself horizontally and vertically

Set the background-repeat property to no-repeat

<style>
Body {
	background-image: url('example_img_girl.jpg');
	background-repeat: no-repeat;
}
</style>

!background cover

<style>
body {
  background-image: url('img_girl.jpg');
  background-repeat: no-repeat;
  background-attachment: fixed; 
  background-size: cover;
}
</style> 

Background-size: cover; to cover
Background-attachment:fixed to fix image from scroll
Background-size: 100% 100%; to stretch
